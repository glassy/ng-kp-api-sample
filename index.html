專案簡介

由於 柯文哲野生官網 所提供的 API 使用範例主要以 jQuery 為主，我將其修改為以 AngularJS 版本的 API 呼叫範例，供 AngularJS 初學者研習之用。

使用方式

  • 本專案的 app.js 檔案中，第 3 行有個 KEY 變數，請先到 柯文哲野生官網 申請 API key 之後，將該變數的值置換成你自己的 API key 即可。如下圖示：
